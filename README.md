# Skanda Website - Local Development

This is a Next.js project for the Skanda website, featuring the engagement models page with interactive components.

## Getting Started

Follow these steps to run the project locally:

1. **Install dependencies**

\`\`\`bash
npm install
# or
yarn install
\`\`\`

2. **Run the development server**

\`\`\`bash
npm run dev
# or
yarn dev
\`\`\`

3. **Open your browser**

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Features

- Interactive engagement models with hover effects
- Responsive design for all screen sizes
- Animated transitions using Framer Motion
- Form submission that sends data to info@skandatech.com

## Project Structure

- `app/` - Next.js app directory with pages and layouts
- `components/` - React components
- `components/ui/` - UI components (button, input, etc.)
- `lib/` - Utility functions

## Technologies Used

- Next.js 14
- React 18
- Tailwind CSS
- Framer Motion
- Lucide React (for icons)
